# Public Assets

## Folder Structure

### Project Images
- `/images/projects/ui-ux/` - UI/UX project screenshots
- `/images/projects/web-apps/` - Web application screenshots
- `/images/projects/video-editing/` - Video project thumbnails
  - `/images/projects/reels/` - Reels thumbnails and covers
- `/images/projects/graphics-design/` - Graphics design work
  - `/images/projects/channel-arts/` - YouTube banners, channel branding
  - `/images/projects/logos/` - Logo designs and brand identity
  - `/images/projects/thumbnails/` - YouTube thumbnails, social graphics

### Other Assets
- `/videos/` - Demo videos and reels
- `/assets/` - Documents, resume, etc.

## Usage
Access files with: `/images/projects/channel-arts/my-banner.jpg`

## Examples
```typescript
// In constants.tsx
thumbnail: '/images/projects/channel-arts/creator-banner.jpg'
thumbnail: '/images/projects/logos/brand-logo.png'
thumbnail: '/images/projects/thumbnails/viral-thumb.jpg'
thumbnail: '/images/projects/reels/fitness-reel.jpg'
```
